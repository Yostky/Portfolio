This weather app is built with all free resourses, and is coded with HTML, CSS, and vanilla Javascript. This app has the capability to display cities with the current and forecasted weather conditions. For every location displayed, there is a high definition background video, a current condition section, and beautiful weather icons for the hourly forecast and daily forecast. Each video displayed changes not only for the specific weather condition, but also for the time of day whether it be day or night. A major limitation for this application is the fact that all of the data is coming from free resources with incomplete databases, meaning that some locations won't display the weather.
